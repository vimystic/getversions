- `pip install pyyaml`
- `python3 -m pip install requests`
- you'll need to auth into github (for rate-limit reasons).
  - go to <https://github.com/settings/personal-access-tokens/new>
  - export the token as `$GITHUB_PAT` in yr terminal or `.zshrc`
